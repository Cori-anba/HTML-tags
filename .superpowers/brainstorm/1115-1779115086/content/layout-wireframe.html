image — 图片标签
img（image）标签为单标签，用于在网页中嵌入图片资源，它是 替换元素，典型场景包括内容插图、图标、Banner等。

📋 属性表格

属性英文原称属性值 / 写法英文解释注意
src ⭐
指定图片的路径/URL source n.来源 URL 或 相对路径 必须指定有效的图片地址
alt ⭐
图片替代文本 alternative n.替代 自定义文本字符串 图片加载失败时显示，对无障碍访问至关重要
width
图片宽度 width n.宽度 像素值（如 400） 建议通过 CSS 设置，保持响应式
