# HTML 标签速查手册 — 设计文档

## 概述

一个纯静态 SPA 网页，用于快速查询 HTML 标签、属性、CSS 属性及全局属性。左侧按字母排序的标签列表 + 顶部搜索栏 + 右侧内容区。暖色护眼配色，支持桌面端和移动端。

## 技术方案

- 纯原生技术栈：HTML + CSS + JS，零依赖，零构建
- SPA 单页面架构：`index.html` + JSON 数据 + JS 动态渲染
- 路由方式：hash 路由（`#tag-name`），如 `#img`、`#input`
- 移动端适配：侧边栏默认隐藏，通过汉堡菜单按钮呼出

## 移动端布局（≤768px）

- 侧边栏默认隐藏，通过导航栏左上角的汉堡菜单按钮控制显示/隐藏
- 侧边栏从左侧滑出，宽度为屏幕宽度的 80%
- 点击遮罩层或选择标签后自动关闭侧边栏
- 内容区占满全宽
- 表格允许横向滚动
- 搜索框宽度缩小

## 表格渲染规则

- 五列：属性/CSS属性 | 英文原称 | 属性值/写法 | 英文解释 | 注意
- 属性名所在行可能占多行（多个属性值时），使用 rowspan 合并
- `star: true` 时在属性名旁边显示 ⭐
- `english` 为空时对应单元格留空
- `note` 为空时对应单元格留空
- 重要关键词在内容中用 `<strong>` 包裹，颜色使用深红强调色

## 搜索逻辑

1. 精确匹配标签名（大小写不敏感）
2. 模糊匹配中文描述
3. 匹配全局属性关键词（class、id、style 等）→ 跳转全局属性页
4. 无匹配时在搜索框下方显示"未找到相关标签"

## 相关标签

- 在标签讲解页底部以卡片形式展示
- 每个卡片为可点击的超链接（hash 路由跳转）
- 卡片样式：米色背景，悬停时变为暖橙色边框

## 交互细节

- 点击侧边栏标签 → 更新 hash → render.js 渲染对应内容
- 搜索框回车 → 更新 hash → 同上
- 页面初次加载时显示首页（可以是简短的欢迎/使用说明页，或者默认显示第一个标签）
- 侧边栏自动滚动到当前选中的标签位置
